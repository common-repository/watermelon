== Description ==

Watermelon is an all-in-one platform for online conversations. One tool to replace all customer service tools. Centralize and automate your online conversations with an intuitive dashboard and a code-free AI chatbot.

This plugin will provide you with Watermelon’s CXhub on your website without any coding knowledge required. Your website visitors can now talk to your business through Livechat or a chatbot right away.

* Be available 24/7
* Reduce calls & mail
* Save support costs

**Why use Watermelon?**
Watermelon’s most user-friendly platform enables you to become a true chatbot builder without the need to code. Besides your website you can also automate and centralize all your customer service channels like WhatsApp, Facebook Messenger, Telegram, Twitter and Gmail. Create the ultimate customer experience or generate more leads with Watermelon:

* Conversation Interface

* Unified Customer profile

* Realtime analytics

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Watermelon settings page to configure the plugin
